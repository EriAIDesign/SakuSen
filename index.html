Create promo videos for the App Store and social media in just a few steps.

About SakuSen

SakuSen is a lightweight iPhone app for making clean promo videos quickly. Choose a video, add text, adjust the layout, and export right away. It is designed for App Store previews, social media posts, and simple promotional clips.

Features

  • Quick workflow: add a video, enter text, and export
  • Multiple export sizes for App Store and social media
  • Adjustable video placement and layout
  • Text editing for each scene
  • Background color and background image customization
  • Simple, clean, distraction-free interface

SakuSen Pro

  • Export without watermark
  • Connect up to 5 scenes
  • Additional customization options
  • Text settings for each scene
